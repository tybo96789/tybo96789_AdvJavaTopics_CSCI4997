Usage:

REUQIRED FILES: config.properties

To start server use the following command:

java -jar ./AtiburcioTyler_HW6.jar s

To start client use the following command:

java -jar ./AtiburcioTyler_HW6.jar c


NOTE: if no arguments are passed it will run the client by default

Server Notes:

Starting the server will host the local properties file

Client notes:

Please use "Select Save Directory" to select where the program will save the received data from the server
Then click "Download File" to download the hosted data from the server and write the contents of data to user specified directory
